# Examples

## `basic`

Minimal file-routing example with:

- `routes/`
- global middleware
- scoped middleware
- scoped error handling
- `hooks.dart`

Run it like this:

```bash
cd example/basic
dart pub get
dart run spry build
dart run .spry/main.dart
```

Or use the CLI serve command:

```bash
cd example/basic
dart pub get
dart run spry serve
```
