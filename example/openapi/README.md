# OpenAPI Example

A focused Spry example that demonstrates:

- `openapi` config in `spry.config.dart`
- document-level `components`
- route-level `openapi`
- route-level `globalComponents`
- generated `public/openapi.json`

Run it like this:

```bash
cd example/openapi
dart pub get
dart run spry build
```

Then inspect `public/openapi.json`.
